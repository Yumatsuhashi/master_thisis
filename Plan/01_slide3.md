# 2 はじめに 背景
>まず初めに研究背景についてお話しします。5GNN以降ではより高速で大容量な通信が求められます
>これを実現するのがビームフォーミングと言う技術ですが,正確なビームフォーミングを実現するためには、正確なCSIが必要です。
>CSIは端末と基地局の間でパイロット信号を送受信してチャネル推定することで行われますが,アンテナ阻止の大規模化により推定のオーバーヘッドが増加したりユーザーの増加よりデータ通信に使える通信リソースが圧迫されるため、スループットが落ちるという課題があります


# 3 CSI予測
>>チャネル予測モデルは、直前に推定した推定値を入力として、次のCSI推定値を予測するよう学習します。
>これにより疎なパイロット信号の送受信でもCSIを獲得することを実現し、リソースグリット、上の通信資源を逼迫することなく、正確なビームフォーミングを実現します


# 4 CSI予測の技術的課題(図を載せたい)
>モデル更新、予測モデルアーキテクチャ高度化、マルチモーダル化、シミュレーションギャップへの対処
>などの研究が行われていますが、自分はこの中でもモデル更新に関して取り組みます
